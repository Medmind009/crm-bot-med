# CRM Bot для медицинских продаж - Спецификация

## Обзор проекта
Улучшенная версия CRM-бота для продаж медицинского оборудования. Бот принимает отчеты от менеджеров через Telegram, парсит их с помощью AI и записывает в Google Таблицы.

## Функционал

### 1. Обработка отчетов
- Прием текстовых отчетов от менеджеров через Telegram
- AI-парсинг для извлечения:
  - Название клиники
  - Контактное лицо и должность
  - ЛПР (лицо, принимающее решение)
  - Телефон
  - Реакция клиента
  - Отправленные материалы
  - Следующий шаг и дата
  - Статус лида
  - Комментарий

### 2. Интеграция с Google Таблицами
- Таблица "Лиды" - основная база клиентов
- Таблица "Касания" - история взаимодействий
- Таблица "Задачи" - задачи для менеджеров

### 3. Улучшения по сравнению с v1
- Гибкая система промптов для AI
- Поддержка нескольких менеджеров
- Уведомления о просроченных задачах
- Команды бота (/start, /help, /stat)
- Логирование и аналитика
- Обработка ошибок

## Архитектура
```
Telegram → Bot API → AI (Gemini) → Google Sheets → Telegram ответ
```

## Конфигурация
- Telegram Bot Token
- Google Sheets API (service account)
- OpenRouter/Gemini API key

## Выходные данные
- JSON с извлеченными данными
- Записи в Google Таблицах
- Сообщение подтверждения в Telegram
