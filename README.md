# 🤖 CRM Бот для медицинских продаж

Улучшенная версия CRM-бота для обработки отчетов менеджеров. Бот принимает отчеты через Telegram, парсит их с помощью AI и записывает в Google Таблицы.

## 📋 Функционал

- ✅ Прием отчетов через Telegram
- ✅ AI-парсинг (Google Gemini через OpenRouter)
- ✅ Запись в Google Таблицы (Лиды, Касания, Задачи)
- ✅ Создание задач для менеджеров
- ✅ Уведомления о статусе
- ✅ Команды /start, /help, /test

## 🚀 Развертывание

### 1. Настройка Google Таблицы

Создайте таблицу с листами:
- **Лиды** (B:Клиника, C:Контакт, D:Должность, E:ЛПР, F:Телефон, G:Email, H:Канал, I:Материалы, J:Реакция, K:Шаг, L:Дата, M:Статус, N:Ответственный, O:Комментарий)
- **Касания** (A:Дата, B:Кто, C:Формат, D:Что произошло, E:Материалы, F:Вопросы, G:Ответы, H:Итог, I:Шаг, J:Комментарий, K:Клиника)
- **Задачи** (A:Клиника, B:Задача, C:Дата создания, D:Дедлайн, E:Ответственный, F:Статус, G:Приоритет)

### 2. Настройка Telegram бота

1. Напишите @BotFather в Telegram
2. Создайте бота командой /newbot
3. Получите токен
4. Настройте webhook

### 3. Настройка переменных окружения

```bash
cp .env.example .env
# Заполните .env своими значениями
```

### 4. Запуск

```bash
# Установка зависимостей
pip install -r requirements.txt

# Запуск
python crm_bot/bot.py
```

### 5. Docker

```dockerfile
FROM python:3.11-slim

WORKDIR /app
COPY . .
RUN pip install -r requirements.txt

CMD ["python", "crm_bot/bot.py"]
```
